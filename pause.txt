‚óŹ Done. Updated speak_from_cursor.sh with:

  How it works:
  1. First tries to read any existing selection/clipboard
  2. If nothing selected, simulates Shift+Ctrl+End to select from cursor to end
  of document
  3. Copies the selection (Ctrl+C)
  4. Speaks the text in background
  5. Restores original clipboard content after

  Platform support:
  - X11: Uses xdotool
  - Wayland: Uses wtype (Sway/Hyprland) or ydotool as fallback
